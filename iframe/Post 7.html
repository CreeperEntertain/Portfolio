Doorway

There is not a lot about this project in particular.
The purpose of this one was mainly to represent someone's vision of a dark hallway for a passion project they were working on. The inspiration was mainly places from games like »CONTROL« mixed with the aesthetic of »Stranger Things«.

The commissioner of this project searched for something with a very dark and evil feeling to it, as well with a lot of mystery. Mostly hidden elements, like a warped hand arising from the water, the seemingly infinitely far up ceiling, as well as the hardly readable objects on the left.
