The Library

Libraries on a basic level are places for reading in a public and silent place, but can be used for many different things. For instance, this library is used to also display the artistic efforts of selected people of a generation, and to display the resources of the place it was built in.

The main goal in visualizing a place like this is simple. You break it down into its most simple elements, figure out how they can each be achieved, and then combine elements like textures or designs from different places. For this project specifically, those were the bookshelves, tables, carpet, and smaller elements like statues.
